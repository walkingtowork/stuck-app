# Stuck App

A decision-making helper app for when you're stuck and don't know what to do.

Built with Vite + React + TypeScript + shadcn/ui + Tailwind CSS.

## Live App

Deployed on Vercel: **https://crepleimstuck.vercel.app**

## Running Locally

```bash
npm install
npm run dev
```

The app will be available at http://localhost:5173.

Other useful commands:

```bash
npm run build    # production build
npm run preview  # preview the production build locally
npm run lint     # run ESLint
```

## Deploying

This project is linked to Vercel (org: `walkingtoworks-projects`). Deployments are triggered manually via the Vercel CLI:

```bash
vercel          # preview deployment
vercel --prod   # production deployment
```

No environment variables are required — the app is fully static.
